# 预先
* 安装了`npm`
* 现在`meteor`不适合windows系统，可以用`linux或者mac`

# 1. 下载安装
* 终端输入 ：`curl https://install.meteor.com/ | sh`

# 2. 找个目录新建项目
*  ` mkdir  meteordemo`
* `meteor create  demo1` 即可新建项目名为`demo1`的meteor项目
*  终端输入 `meteor` 即可运行一个`web应用`啦。

# 3.打开浏览器 访问 `localhost:3000`
